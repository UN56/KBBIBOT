# KBBIBOT
 KBBI Discord Bot

## Usage

clone repo  
`git clone https://github.com/UN56/KBBI.git`

`cd KBBI`

run  
`pip install -r requirements.txt`

set up your bot on [Discord developer portal](https://discord.com/developers/)

create environment variable  
`API_KEY = YOURTOKEN`

run  
`python main.py`
